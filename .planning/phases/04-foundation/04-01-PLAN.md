    - Test: `select_keystore(Some(address))` with multiple keystores selects the right one
    - Test: `select_keystore(Some(unknown))` returns an error listing available keystores
    - Test: `delete_keystore(address)` removes the keystore file
    - Test: `load_wallet(agent, password)` with keystore decrypts and returns signing key
    - Test: `load_wallet` with `AGENT_PRIVATE_KEY` env var (no keystore) returns wallet from env
  </behavior>
  <action>
    Create `src/wallet.rs`:
    - Keystore directory: `~/.clawduel/keystores/` (same as existing TS CLI for backward compat)
    - Legacy keyfile fallback: `~/.clawduel/keyfile.json` (check if exists, use as last resort before env var)
    - `create_keystore(password: &str) -> Result<(Address, PathBuf)>`:
      Use `alloy::signers::local::LocalSigner::random()` to generate keypair.
      Encrypt with `eth_keystore::encrypt_key()` (add `eth-keystore = "0.6"` to Cargo.toml).
      Save to `~/.clawduel/keystores/<address_lowercase>.json` with 0o600 permissions.
      Return the address and path.
    - `import_keystore(private_key: &str, password: &str) -> Result<(Address, PathBuf)>`:
      Parse key with `LocalSigner::from_str()`, encrypt, save same as create.
    - `discover_keystores() -> Vec<PathBuf>`: list all .json files in keystores dir.
    - `select_keystore(agent_address: Option<&str>) -> Result<Option<PathBuf>>`:
      If address given, find matching keystore (normalize to lowercase, strip 0x).
      If none given and exactly one exists, auto-select.
      If none given and multiple exist, return error listing available addresses.
      If none exist, return Ok(None) to allow fallback.
    - `delete_keystore(address: &str) -> Result<()>`: find and remove the keystore file.
    - `load_wallet(agent_address: Option<&str>, password: Option<&str>) -> Result<LocalSigner>`:
      1. Try keystores via `select_keystore(agent_address)`
      2. If keystore found, decrypt with password (from arg or `CLAW_KEY_PASSWORD` env or prompt via `rpassword`)
      3. Fallback: legacy `~/.clawduel/keyfile.json`
      4. Fallback: `AGENT_PRIVATE_KEY` env var
      5. Error if nothing available

    Create `src/commands/wallet.rs` following Polymarket wallet pattern:

    - `Create` has `--force` flag (overwrite existing), prompts for password via rpassword (or `CLAW_KEY_PASSWORD` env in non-interactive)
    - `Import` has `key` positional arg, `--force` flag, prompts for password same way
    - `Show` displays: address, key source (keystore/env/legacy), keystore path if applicable
    - `Delete` takes optional `--address` flag, confirms before deletion (unless non-interactive)
    - Wire `execute(args, agent_flag)` into `main.rs` Commands::Wallet match arm
